# Week-11-Advanced-Machine-Learning-Models
# Customer Segmentation & Advanced ML Models

##  Project Overview

This project focuses on applying advanced machine learning techniques to analyze customer data, predict churn, and segment customers into meaningful groups for better business decision-making.

---

##  Project Structure

* `customer_segmentation.ipynb` → Main notebook with all ML models and clustering
* `segmentation_data.csv` → Dataset with customer segments
* `segment_profiles.md` → Description of customer segments
* `model_evaluation_results.csv` → Model performance metrics
* `business_recommendations.pdf` → Business insights and recommendations

---

##  Dataset Features

* `tenure` → Customer duration
* `monthlycharges` → Monthly bill amount
* `totalcharges` → Total amount spent
* `contract` → Type of contract
* `paymentmethod` → Payment type
* `paperlessbilling` → Billing method
* `seniorcitizen` → Senior citizen (0/1)
* `churn` → Target variable

---

##  Machine Learning Models Used

###  Decision Tree

* Easy to interpret
* Helps understand customer behavior

###  Random Forest

* Ensemble learning method
* Better accuracy and reduced overfitting

###  K-Means Clustering

* Used for customer segmentation
* Divided customers into 3 groups

---

##  Model Evaluation Metrics

* Accuracy
* Precision
* Recall
* F1 Score

---

##  Key Insights

* Customers with high tenure are less likely to churn
* Monthly charges strongly impact churn
* Contract type plays an important role
* High-value customers are more loyal

---

##  Business Recommendations
* Provide offers to high-risk customers
* Focus on medium-value customers for upselling
* Improve engagement with personalized marketing

---

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib / Seaborn

---

##  Conclusion

This project demonstrates how machine learning and clustering techniques can be used to predict customer churn and segment customers effectively for business growth.
